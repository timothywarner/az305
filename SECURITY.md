# Security Policy

## Reporting a Vulnerability
If you discover a security issue in this repository, please report it privately by emailing Tim Warner at tim@techtrainertim.com.

When reporting, please include:
- A clear description of the issue
- Steps to reproduce (if applicable)
- Potential impact

We will acknowledge receipt within 3 business days and provide a remediation plan or status update within 10 business days.

## Supported Versions
This repository contains training materials rather than production software. Security fixes are applied to the latest content on the default branch.
