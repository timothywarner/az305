# Contributing to AZ-305 Study Resources

Thanks for your interest in improving this repository! Contributions that help learners prepare for the AZ-305 exam are welcome.

## Ways to Contribute
- Fix typos, broken links, or formatting issues
- Add or update study resources
- Improve lab guidance or scripts
- Enhance documentation for clarity

## Before You Start
- Please read the Code of Conduct.
- Search existing issues to avoid duplicates.

## How to Contribute
1. Fork the repository and create a feature branch.
2. Make your changes with clear, concise commits.
3. Open a pull request and fill out the template.

## Content Guidelines
- Keep content aligned with the AZ-305 exam objectives.
- Prefer authoritative sources (Microsoft Learn, official docs, vendor labs).
- Avoid posting copyrighted or paywalled content.

## Questions or Help
If you're unsure where to start or have questions, contact Tim Warner at tim@techtrainertim.com.
